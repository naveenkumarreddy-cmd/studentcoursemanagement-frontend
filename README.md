## Overview

The **Student Course Management Frontend** is a responsive web application built using **React.js** that provides an intuitive interface for managing students, instructors, courses, and enrollments. It enables role-based dashboards for Admin, Instructor, and Student users, allowing efficient interaction with backend services through secure REST API integration.

The application focuses on usability, performance, and clean UI design to provide a seamless course management experience.

---

## Features

* Role-based dashboards (Admin, Instructor, Student)
* Course creation, editing, and deletion
* Student and instructor management interfaces
* Course enrollment functionality
* Secure API integration using Axios
* Responsive and user-friendly UI
* Protected routes for authenticated users

---

## Tech Stack

* **Frontend Framework:** React.js
* **Language:** JavaScript (ES6+)
* **HTTP Client:** Axios
* **Styling:** CSS3 / Responsive Design
* **Routing:** React Router

## Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/student-course-management-frontend.git
```

### 2. Navigate to the project directory

```bash
cd student-course-management-frontend
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the application

```bash
npm start
```

The application will start at:

```
http://localhost:3000
```
